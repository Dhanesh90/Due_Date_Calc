# Due Date Calculator & Pregnancy Companion

A beautiful, glassmorphism-style web application that calculates pregnancy due dates and provides a detailed week-by-week timeline of the pregnancy journey.

## Features

- **Accurate Calculation**: Uses Naegele's rule to estimate due dates based on LMP and cycle length.
- **Weekly Insights**: Automatically generates a 40+ week timeline.
- **Smart Highlighting**: Detects the current pregnancy week based on today's date and highlights it ("YOU ARE HERE").
- **Detailed Content**: Provides development milestones, medical tests, and common symptoms for every week.
- **Cute Aesthetic**: Soft pastel colors, cute baby/embryo illustrations, and a responsive design.

## How to Run Locally

You can run this project directly in any modern web browser.

1.  Download the project files.
2.  Double-click `index.html` to open it in your browser.
    *   *No server or installation required!*

## File Structure

- `index.html`: Main application structure.
- `style.css`: All styling, animations, and responsive design rules.
- `script.js`: Logic for date calculation and dynamic timeline generation.
- `data.js`: Content database for weekly pregnancy details.
- `assets/`: Contains images and icons.
